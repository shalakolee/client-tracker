# Project Brief

## Project
ClientTracker (MAUI app)

## Goals
- Deliver a polished, Material 3-aligned mobile UX across all pages.
- Support Android builds/deploys from the repo.
- Keep navigation, lists, forms, and sheets aligned with M3 guidance.

## Scope
- MAUI client app only.
- Android deploys via `dotnet build ... -t:Run -f net10.0-android`.

## Non-Goals
- Backend API work unless explicitly requested.
- Adding external UI frameworks beyond current MAUI setup unless asked.
